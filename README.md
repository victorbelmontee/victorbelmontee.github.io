# Descripción del Formulario de Contacto

## Campos de Formulario

- **Nombre**
  - **Tipo:** `text`
  - **Descripción:** Permite introducir el nombre del usuario. Se asocia a la etiqueta `<label>` mediante el atributo `for="name-field"` para mejorar la accesibilidad.

- **Contraseña**
  - **Tipo:** `password`
  - **Descripción:** Permite ingresar una contraseña de forma segura, ocultando los caracteres introducidos.

- **Fecha de Nacimiento**
  - **Tipo:** `date`
  - **Descripción:** Permite seleccionar la fecha de nacimiento utilizando un selector de fecha nativo del navegador.

- **Color Favorito**
  - **Tipo:** `color`
  - **Descripción:** Proporciona un selector de color que permite al usuario elegir su color favorito.

- **Edad**
  - **Tipo:** `number`
  - **Descripción:** Permite ingresar la edad del usuario. Se puede configurar con un valor por defecto, por ejemplo de **18** y establecer límites con `min="1"` y `max="120"`.

- **Suscripción (Checkbox)**
  - **Tipo:** `checkbox`
  - **Descripción:** Campo de selección que nos permite checkear una caja.

## Botón de Envío

- **Enviar**
  - **Tipo:** `submit`
  - **Descripción:** Botón que envía los datos del formulario para su posterior procesamiento.
